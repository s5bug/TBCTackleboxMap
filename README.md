# TBCTackleboxPlugin

Edit `TBCTackleboxPlugin.csproj.user` correcting the path to your The Big Catch Tacklebox folder. An example
with Windows Steam on the C drive is shown below:

```msbuild
<Project>
    <PropertyGroup>
        <TbcTackleboxPath>C:/Program Files (x86)/Steam/steamapps/common/The Big Catch Tacklebox</TbcTackleboxPath>
    </PropertyGroup>
</Project>
```
